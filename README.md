# AASHVI INTERN - Internship & Training Platform

A premium, modern internship and training platform built with React, Tailwind CSS, Node.js, and MongoDB.

## 🚀 Key Features
- **Modern UI**: High-fidelity design with glassmorphism and smooth animations.
- **Responsive**: Fully optimized for mobile, tablet, and desktop.
- **Search & Filter**: Real-time internship and course discovery.
- **Certificate Verification**: Secure validation of program credentials.
- **Student Dashboard**: Track applications and course progress.
- **Auth System**: Integrated Login and Signup (JWT ready).

## 🛠 Tech Stack
- **Frontend**: React 18, Vite, Tailwind CSS, Framer Motion, Lucide Icons.
- **Backend**: Node.js, Express, MongoDB, Mongoose.
- **Other**: JWT for Auth, Dotenv for config.

## 📦 Folder Structure
- `/client`: Frontend React application.
- `/server`: Backend Express API and MongoDB models.

## 🛠 Installation & Setup

### 1. Clone the repository
```bash
git clone <your-repo-link>
cd AASHVI-INTERN
```

### 2. Setup Server
```bash
cd server
npm install
# Create a .env file and add your MONGODB_URI and JWT_SECRET
npm run dev
```

### 3. Setup Client
```bash
cd ../client
npm install
npm run dev
```

## 🌐 Deployment

### Frontend (Vercel)
1. Push `client` code to GitHub.
2. Link Vercel to your repository.
3. Set the Root Directory to `client`.
4. Deploy.

### Backend (Render / Railway)
1. Push `server` code to GitHub.
2. Create a new Web Service.
3. Set base directory to `server`.
4. Add environment variables.
5. Deploy.
